Mise en place d'un site de Masterclass

Le Défi

Permettre aux clientes de Madame Y de s'inscrire en ligne à une masterclass, de choisir un créneau horaire et de recevoir des emails de rappel automatisés pour ne pas manquer l'événement.

Ma Solution Technique

J'ai déployé un site web sur une infrastructure que j'ai personnellement configurée pour garantir performance et stabilité :

  • Déploiement sur serveur privé (VPS) pour un contrôle total de l'environnement.
  • Gestion de version avec Git pour un suivi rigoureux du code.
  • Configuration de Nginx comme reverse proxy pour la sécurité et la gestion du trafic.
  • Utilisation de PM2 pour assurer que l'application reste en ligne en continu.

Lien avec le poste de Data Scientist

Cette expérience démontre des compétences fondamentales en DataOps/MLOps :

  • ✅ La mise en place de cette infrastructure est la base requise pour déployer des applications data ou des API de modèles de Machine Learning.
  • ✅ La méthode de déploiement illustre ma maîtrise des outils de la chaîne CI/CD (Git, serveur Linux), essentiels pour l'industrialisation des projets d'IA.
  • ✅ La gestion des créneaux et des rappels est une problématique de gestion de données temporelles.
