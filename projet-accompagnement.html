Création d'un site d'accompagnement avec paiement

Le Défi

Créer un site web permettant aux clientes de souscrire à des offres d'accompagnement en intégrant une solution de paiement en ligne totalement sécurisée, simple d'utilisation et fiable.

Ma Solution Technique

J'ai développé une application web avec un backend dédié à la gestion des paiements :

  • Backend en Node.js pour communiquer de manière sécurisée avec des services externes.
  • Intégration de l'API Stripe, une référence mondiale pour le paiement en ligne.
  • Architecture sécurisée où les données de carte bancaire ne transitent jamais par mon serveur, mais sont gérées directement par Stripe.

Lien avec le poste de Data Scientist

Ce projet met en avant des compétences techniques et conceptuelles clés.

  • ✅ La création d'un backend et l'intégration d'API sont des compétences directement transposables à l'environnement Python avec des frameworks comme Flask ou FastAPI pour créer des API de Machine Learning.
  • ✅ Il prouve ma capacité à manipuler des flux de données sensibles (transactionnelles), un enjeu majeur dans le secteur bancaire.
  • ✅ La maîtrise de Git a été essentielle pour gérer à la fois le code du front-end et du back-end.
