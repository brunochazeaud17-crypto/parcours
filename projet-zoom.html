Formulaire d'inscription Zoom

Le besoin : Automatiser la collecte de données

La gestion manuelle des inscriptions était chronophage et source d'erreurs. L'objectif était de créer un pipeline de données entièrement automatisé pour capturer, structurer et exploiter les informations des nouveaux inscrits, un cas d'usage typique en préparation de données pour le Machine Learning.

La solution technique : Un pipeline ETL "low-code"

J'ai conçu un workflow d'automatisation (ETL : Extract, Transform, Load) avec l'outil n8n, démontrant une approche agile pour la résolution de problèmes data :

  • Extract : Un webhook capture en temps réel les données soumises depuis le formulaire du site web.
  • Transform : Le workflow valide et formate les données pour garantir leur propreté et leur cohérence.
  • Load : Les données structurées sont chargées via API dans une base de données (Airtable), créant une source de données fiable et prête à être analysée ou utilisée pour entraîner un modèle.

Lien avec l'offre d'emploi

Ce projet illustre ma capacité à concevoir des flux de données efficaces, une compétence clé pour un Data Scientist :

  • ✔️ DataOps : J'ai mis en place un pipeline de données automatisé, de la source à la destination, illustrant les principes fondamentaux du DataOps.
  • ✔️ Méthodologies Agiles : L'utilisation d'un outil "low-code" a permis de livrer rapidement une solution fonctionnelle, démontrant une approche itérative et agile.
  • ✔️ Maîtrise de l'environnement Python : Bien que n8n soit "low-code", il permet d'intégrer des scripts Python pour des transformations de données plus complexes, une compétence que je maîtrise pour des cas d'usage avancés.
  • ✔️ Équipe pluridisciplinaire : Ce pipeline est le pont entre les besoins métiers (marketing) et les équipes techniques (Data & AI Engineers), montrant ma capacité à collaborer efficacement.
